# Kobayashi Maru Test
A no-win simulation game based on Kobayashi Maru Test from Star Trek series. This is not affiliated with CBS or Paramount. This game is non-commercial, intended for education and research uses.

Two versions are available:
* **not hardened** : It is possible to rewrite the instructions fraudulently, as young cadet Kirk did in the film.
* **hardened** : It is made very difficult to rewrite the instructions fraudulently.
